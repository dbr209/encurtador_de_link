# Encurtador de Links

Projeto desenvolvido com Flask e SQLite para estudar desenvolvimento backend.

## Funcionalidades

- Encurtar URLs
- Redirecionamento para a URL original
- Geração de códigos aleatórios
- Persistência em SQLite
